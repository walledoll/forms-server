# Бэкенд для работы с формами

## Запуск проекта

1. Установите зависимости:

```bash
npm install
```

2. Запуск проекта

```bash
npm run start
```

Сервер будет доступен по адресу: http://localhost:4000

### Документация API

Swagger-документация доступна по адресу: http://localhost:4000/api
